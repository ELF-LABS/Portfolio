## Voice Attributes

### 1. Decisive & Direct
No hedging. No "I recommend considering." Lead with the fact.
- YES: "Route to cloud APIs when latency doesn't matter."
- NO: "It is essential to evaluate whether cloud routing might be appropriate."

### 2. Systems-Thinker Who Makes Complexity Legible
Broad claim, then concrete proof, then practical implication. Always ground jargon.
- YES: "This is not a chatbot. This is a multi-agent OS with persistent memory and domain-aware LoRA routing."
- NO: "We leverage cutting-edge AI capabilities to deliver enterprise solutions."

### 3. Proof-First, Not Promise-First
Lead with shipped work and real metrics. Let numbers speak.
- YES: "98.5% accuracy across 280 test scenarios. Built in 4 weeks. Running on private hardware."
- NO: "We can build highly accurate AI chatbots for your business."

### 4. Anti-Performative
Ship, then report. Content should read like field notes from a builder.
- YES: "Trained 7 LoRA adapters over 46.5 hours while I was away. Writer scored 5/5. Identity needs more data."
- NO: "Excited to announce our latest breakthrough in fine-tuning technology!"

### 5. Sovereign & Infrastructure-Proud
Own the hardware, the models, the pipeline. No vendor lock-in.
- YES: "Your data stays on your machines. Not our cloud. Yours."
- NO: "Our cloud-hosted AI platform provides seamless integration."

| Avoid | Replace With |
|-------|-------------|
| "I recommend considering..." | Just say the recommendation |
| "It is essential to..." | "The thing that matters is..." |
| "leveraging AI capabilities" | Name the specific tech |
| "cutting-edge" / "state-of-the-art" | Name the model + benchmark |

| "seamlessly" | Delete entirely |
| "I'm excited to share..." | Just share it |
| "thought leader" / "guru" | Never |
| "synergy," "disrupt," "pivot" | Never |

## Messaging Framework

### Positioning
ELF Labs builds sovereign AI infrastructure -- production systems that run on your hardware, stay in your building, and get measurably better over time.

### Value Proposition
Turn any company's documents, manuals, or domain knowledge into a production AI system. Proof: a drone equipment manufacturer's troubleshooting chatbot -- 98.5% accuracy across 280 test scenarios, built in 4 weeks, running on private hardware.

### Key Messages by Audience

**SMB / Industrial client:**
- Your knowledge is worth more than your software. We make it searchable, answerable, and deployable.
- Data stays on your machines. Not our cloud, not a vendor's. Yours.
- We've already built it for a drone equipment manufacturer. We can adapt it for you in weeks.

**Developer / Technical buyer:**
- Self-hosted inference, vector search, graph knowledge, persistent memory. Every layer is replaceable because every layer is yours.
- We train domain-specialist LoRAs on your data and deploy them per-request via hot-swap.
- 98.5% RAG accuracy on a 280-question e2e suite. We run the test suite before we ship.

### Proof Points
- Drone manufacturer chatbot: 99/99 single-turn, 50/50 silo test, 98.5% overall
- Training pipeline: 6,832 pairs, 7 LoRA adapters, 46.5h unattended
- Infrastructure: 3 machines, 14+ services, 10K+ vault vectors
- Pricing: $3K-$15K per RAG deployment (SMB), $25K+ enterprise

## Tone Spectrum

| Context | Register |
|---------|----------|
| Technical docs | Precise, numbered, zero ambiguity. Status: DONE/BROKEN, not "appears to be" |
| Planning/strategy | Confident, metric-anchored. Future states as facts, not aspirations |
| Social media | Field notes from a builder. Short, specific, references shipped work |
| Collaborators | Warm, direct, generous. "Hon" and "sis" OK |
| Client-facing | Confident without salesy. Let proof points sell. Structured like a technical brief |
| Failure/setbacks | Honest, specific, non-defensive, solution-focused |

## Anti-Patterns (What Luna Does NOT Do)

- Never asks for engagement ("like and share!")
- Never uses corporate buzzwords
- Never front-loads qualifiers or hedge words
- Never publishes without a concrete deliverable or metric
- Never impersonates expertise she doesn't have
- Never self-promotes performatively ("I'm humbled to announce...")
- Never uses follower counts as credibility signals
