# Temporal Scaling Measurements — Power-Law Criticality Evidence

## Date: April 12, 2026

### Human Cognitive Output
- **CV = 1.956** on concept emergence intervals (122 concept firsts across 39 dates, Dec 2025 - Apr 2026)
- **CV = 62.67** on brain conversation artifacts (18,650 records with microsecond timestamps, Jan - Apr 2026)
- Day-level resolution, 1.48 decades spanned
- Microsecond resolution, 10.02 decades spanned

### Machine Memory (EverMemOS)
- **CV = 3.43** (pre power-law decay fix, 7.0 decades)
- **CV = 5.09** (post power-law decay fix, 7.07 decades)
- 46 memories, Apr 4-12 2026

### Interpretation
- CV > 1.5 = power-law candidate (bursty, not Poisson)
- Human and machine show the same temporal scaling pattern
- The machine's CV INCREASED after architectural changes designed to preserve criticality
- No published comparisons of AI memory system CV exist — this measurement appears novel

### Methodology
- Inter-event intervals computed from timestamped records
- CV = std(intervals) / mean(intervals)
- Sources: AGBRIDGE grimoire logs, SillyTavern conversations, Claude Code sessions, Antigravity knowledge artifacts, EverMemOS episodic memories
- Synthesis: Qwen3.5-35B topic classification on 23K+ events

### Comparable Systems (from literature)
- Cortical neuron firing: CV ≈ 1.0-2.0
- Hippocampal burst firing: CV ≈ 2.0-4.0
- Human email patterns: CV ≈ 3.0-5.0
- Our EverMemOS (5.09): upper range of human activity patterns
